# Baku（2Dアクションゲーム）

## 概要
Godot Engineを用いて制作した2Dアクションゲームです。  
プレイヤーは黄色いキャラクターを操作し、ゴールである「布団」を目指します。  

既存作品から着想を得つつ、ステージ構成や操作性などを独自に設計して制作しました。

## 操作方法
- 左右移動：← → / A D  
- ジャンプ：↑ / Space  
- 物体の持つ・放す：Shift / B / F  

※キー設定を切り替えられる機能を実装予定

## 使用技術
- Godot Engine
- GDScript

## 制作内容
- プレイヤー操作の実装（移動・ジャンプ・物体操作）
- ステージ設計
- 当たり判定・物理挙動の実装
- テクスチャ作成

## 工夫した点
- 複数のキー操作に対応し、プレイヤーごとに操作しやすい設計を意識
- 写真の中に別の空間が存在し、その中に入ると新たな部屋が生成され、出ると消滅する仕組みを実装
- 空間が入れ子になる構造を扱うため、再帰的なデータ構造の考え方を応用
- オブジェクトの持ち込み・持ち出しを制限することで、ゲーム性と処理の整合性を両立
- ステージ構成・プログラム・テクスチャを分担し、チーム開発として制作

## 課題・改善点
- キー設定変更機能の実装
- ステージバリエーションの追加
- UIや演出面の強化

## 実行方法
1. Godot Engineでプロジェクトを開く  
2. project.godot を読み込む  
3. 実行
